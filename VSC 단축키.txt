VSC 단축키


1. ctrl + D -> 자주 사용되는 코드를 가져다가 누르며는 똑같이 생긴 녀석들을 전부 선택하게 해줌 개편함 ㅋ

2. alt + click -> 원하는 위치에 전부 커서를 둘 수 있음.

3. alt 위 / 아래 방향키 -> 원하는 코드들을 위아래로 움직일 수 있음. -> 위치 변경가능
 
4. alt  + Shift 위 / 아래 방향키 -> 3.에서 복사하는 거임 움직이는게 아니라.

5. ctrl + / -> VSC는 우리가 코딩하는 언어를 파악해서 그에 맞는 주석처리를 해준다.

6. alt + Shift + I => 선택한 영역에 커서가 잔뜩 생김   와우 이건 진또 배기다 ㅋ(끝부분에 생김 ㅇㅇ)

7. alt + Shift + Drage => 드래그 하는 영역에 마우스 커서가 생긴다 이건 또 엄청나구먼 ㅋ
